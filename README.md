### 第一个go程序(helloworld.go)

- helloworld程序编写
- 运行命令`go run filename.go`
- { 不能放在单独一行

### 格式化输出(output_formatting.go)

- `%s: 字符串`
- `%d: 整数`
- `%f: 浮点数`
- `%t: 布尔值`
- `%c: 字符`
- `%v: 默认值`

### 斐波那契数列(Fibonacci.go)

- 并行赋值是先计算右侧再赋值，循环计算需要将a+b独立计算

### 显式转换(explicit_conversion.go)

- 两个类型名称不完全一致的场景，必须要**显式类型转换**。
